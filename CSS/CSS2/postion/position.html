Postion property

Position - Static

By default the postion property is set to static and the images will look like this...

Position - Fixed
This will not move during any movement, bcoz its position is fixed.

Position - Relative

In Relative positions the element are positioned relative to its normal postion.The image2 has position relative, with top property 200px and left property 100px .
Also if you notice the original postion gap of the image is intact ..

Position - Absolute

Absolute property is used to overlaps elements.
Also if you notice the original postion gap of the image is not intact ..
